## Get started

1. Install Orla Code:

**MacOS/Linux:**

```bash
curl -fsSL https://Orla.ai/install.sh | bash
```

**Homebrew (MacOS):**

```bash
brew install --cask Orla-code
```

**Windows:**

```powershell
irm https://Orla.ai/install.ps1 | iex
```

**NPM:**

```bash
npm install -g @anthropic-ai/Orla-code
```

NOTE: If installing with NPM, you also need to install [Node.js 18+](https://nodejs.org/en/download/)

2. Navigate to your project directory and run `Orla`.
